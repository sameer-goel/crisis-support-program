# 🕊️ Crisis Support Program

**Problem:** During crisis, so many need help and so many want to provide help. How can we match the right people who can help each other?

**Analysis:** It's a supply and demand problem.

**Solution:** Crisis-Support platform powered by an intelligent matching engine.
